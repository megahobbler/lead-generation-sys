# SWOPLABS Lead Generation System
Automated lead generation and cold email outreach for SWOPLABS.AI

## System Overview
- 6 AI agents working together
- Automated lead scraping, email writing, campaign management
- Built-in safety checks and human oversight
- Complete audit trail and logging
- Loop prevention and error handling

## Agents
1. Lead Scout - Finds qualified restaurants/clinics
2. Email Writer - Creates personalized cold emails (4 A/B test templates)
3. Campaign Manager - Sends emails, tracks engagement
4. Analytics Tracker - Measures performance, A/B test results
5. Insights Generator - Analyzes data, provides recommendations
6. Oversight - Monitors all agents 24/7, detects anomalies

## Safety Features
- Never/Always rules enforced
- Escalation triggers for human approval
- Loop detection (prevents infinite runs)
- Complete execution logging (never deleted)
- Daily consolidated reports

## Quick Start
1. Open dashboard: /Users/Sean/SWOPLABS/dashboard.html
2. Check system status
3. Review pending approvals (if any)
4. Monitor execution logs

## Files
- `/config/` - System configuration
- `/data/` - Leads, emails, tracking data
- `/logs/` - Complete execution history
- `/analytics/` - A/B test results, insights
- `/oversight/` - System health reports
